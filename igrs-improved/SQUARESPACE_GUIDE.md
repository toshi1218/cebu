# igrs.online SEO設定手順書（Squarespace Personal プラン版）

## 概要

このガイドでは、Squarespace Personalプランで実施可能なSEO改善を解説します。

**所要時間**: 約30〜45分

---

## Step 1: Squarespace管理画面にログイン

1. https://www.squarespace.com にアクセス
2. 右上の「LOG IN」をクリック
3. メールアドレスとパスワードでログイン
4. igrs.onlineのサイトを選択

---

## Step 2: サイト全体のSEO設定

### 2.1 サイトタイトルとタグラインの設定

1. 左メニュー「**設定**」（Settings）をクリック
2. 「**サイトの可用性**」または「**一般**」（General）をクリック
3. 以下を設定：

| 項目 | 設定値 |
|------|--------|
| **サイトタイトル** | 株式会社IGRS |
| **タグライン** | フィリピン公的書類取得・アポスティーユ認証代行サービス |

4. 「**保存**」をクリック

### 2.2 SEO基本設定

1. 左メニュー「**設定**」→「**SEO**」をクリック
2. 以下を確認・設定：

| 項目 | 設定 |
|------|------|
| **検索エンジンにサイトを表示** | オン ✅ |
| **SEOタイトル形式** | %p - %s（ページタイトル - サイト名） |

---

## Step 3: 各ページのSEO設定

### 3.1 ホームページ（トップページ）

1. 左メニュー「**ページ**」（Pages）をクリック
2. 「**ホーム**」ページの横にある「**⚙️**」（歯車アイコン）をクリック
3. 「**SEO**」タブを選択
4. 以下を入力：

**SEOタイトル（50〜60文字）:**
```
株式会社IGRS | フィリピン公的書類取得・アポスティーユ認証代行
```

**SEO説明文（120〜160文字）:**
```
フィリピンの出生証明書・婚姻証明書・無犯罪証明書(NBI)・独身証明書(CENOMAR)などの公的書類取得を代行。DFAアポスティーユ認証もワンストップで対応。セブ島営業所から最短3〜4週間で取得可能。
```

5. 「**保存**」をクリック

### 3.2 サービスページ

1. 「**サービス**」ページの「**⚙️**」をクリック
2. 「**SEO**」タブで以下を入力：

**SEOタイトル:**
```
サービス一覧 | フィリピン公的書類取得代行 - 株式会社IGRS
```

**SEO説明文:**
```
株式会社IGRSが提供するフィリピン公的書類取得代行サービスの一覧。出生証明書、婚姻証明書、CENOMAR、NBI無犯罪証明書、LTO運転経歴証明書、DFAアポスティーユ認証に対応。
```

### 3.3 料金ページ

**SEOタイトル:**
```
料金表・費用一覧 | フィリピン公的書類取得代行 - 株式会社IGRS
```

**SEO説明文:**

株式会社IGRSのフィリピン公的書類取得代行サービスの料金表。出生証明書、婚姻証明書、CENOMAR、NBI、LTO、DFAアポスティーユの費用を明確に表示。追加料金なしの安心価格。
```

### 3.4 会社情報ページ

**SEOタイトル:**
```

**SEO説明文:**

### 3.5 お問い合わせページ

**SEOタイトル:**

### 3.6 プライバシーポリシーページ

## Step 4: ソーシャル共有設定（OGP代替）

Personalプランでもソーシャル共有画像は設定できます。

### 4.1 サイト全体のソーシャル画像

1. 左メニュー「**設定**」→「**ソーシャル共有**」または「**Social Sharing**」
2. 「**ソーシャル共有ロゴ/画像**」に画像をアップロード

**画像仕様:**
- サイズ: 1200 x 630 px
- フォーマット: PNG または JPG
- 内容: 会社ロゴ + キャッチコピー

### 4.2 各ページのソーシャル画像（オプション）

1. 各ページの「**⚙️**」→「**ソーシャル画像**」タブ
2. ページ専用の画像をアップロード（任意）

---

## Step 5: ソーシャルリンクの設定

### 5.1 SNSアカウントの連携

1. 左メニュー「**設定**」→「**ソーシャルリンク**」
2. 「**リンクを追加**」をクリック
3. 以下を追加：

| プラットフォーム | URL |
|-----------------|-----|
| X (Twitter) | https://twitter.com/GswCnxL7Sg15778 |

4. 「**保存**」をクリック

---

## Step 6: サイトマップの確認

Squarespaceは自動的にサイトマップを生成します。

### 6.1 確認方法

ブラウザで以下にアクセスして確認：
```
https://igrs.online/sitemap.xml
```

ページ一覧が表示されればOKです。

---

## Step 7: Google Search Consoleへの登録

### 7.1 Search Consoleにアクセス

1. https://search.google.com/search-console にアクセス
2. Googleアカウントでログイン

### 7.2 プロパティを追加

1. 左上の「**プロパティを追加**」をクリック
2. 「**URLプレフィックス**」を選択
3. `https://igrs.online` を入力
4. 「**続行**」をクリック

### 7.3 所有権の確認（Squarespace用）

1. 確認方法で「**HTMLタグ**」を選択
2. 表示されたメタタグをコピー
   ```html
   <meta name="google-site-verification" content="xxxxxxxx" />
   ```
3. Squarespaceに戻る
4. 「**設定**」→「**SEO**」→「**検索エンジンの確認**」
5. 「**Google Search Console**」の欄に確認コードを貼り付け
6. 「**保存**」
7. Search Consoleに戻り「**確認**」をクリック

### 7.4 サイトマップを送信

1. 左メニュー「**サイトマップ**」をクリック
2. 「新しいサイトマップの追加」に `sitemap.xml` と入力
3. 「**送信**」をクリック

---

## Step 8: 設定完了後の確認

### 8.1 SEO確認

各ページをシークレットモードで開き、タイトルと説明文が正しく表示されるか確認

### 8.2 ソーシャル共有テスト

**Facebook/LINEデバッガー:**
https://developers.facebook.com/tools/debug/

1. `https://igrs.online` を入力
2. 「**デバッグ**」をクリック
3. タイトル、説明、画像が表示されるか確認

## Businessプランへのアップグレードを検討する場合

以下の機能が追加で利用可能になります：

| 機能 | 効果 |
|------|------|
| Code Injection | 構造化データ（JSON-LD）追加可能 |
| 高度なOGP設定 | 全SNSで最適な表示 |
| カスタムCSS | デザインの細かい調整 |
| 広告非表示 | Squarespaceバッジ削除 |

**アップグレード手順:**
1. 「**設定**」→「**サイトの可用性**」→「**プラン**」
2. 「**Business**」を選択
3. 支払い情報を入力
