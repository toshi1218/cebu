# OGP画像作成仕様書

## 基本仕様

| 項目 | 値 |
|------|-----|
| **ファイル名** | ogp.png |
| **サイズ** | 1200 x 630 px |
| **フォーマット** | PNG または JPG |
| **ファイルサイズ** | 300KB以下推奨 |

## デザイン要件

### レイアウト

```
+--------------------------------------------------+
|                                                  |
|     [ロゴ]                                        |
|                                                  |
|     株式会社IGRS                                  |
|                                                  |
|     フィリピン公的書類取得                          |
|     アポスティーユ認証代行サービス                   |
|                                                  |
|     [アイコン: 書類] [アイコン: 飛行機] [アイコン: 認証]|
|                                                  |
|     igrs.online                                  |
|                                                  |
+--------------------------------------------------+
```

### カラースキーム

| 用途 | カラーコード | 説明 |
|------|------------|------|
| **背景** | #1A5F7A | プライマリカラー（青緑系） |
| **グラデーション** | #1A5F7A → #2D8FAA | 左上から右下へ |
| **テキスト（主）** | #FFFFFF | 白 |
| **アクセント** | #FFD700 | ゴールド（強調用） |

### フォント

- **会社名**: 太字、48px以上
- **キャッチコピー**: 中太字、32px
- **URL**: 細字、24px

### 注意事項

1. **セーフエリア**: 端から50px以上のマージンを確保
2. **テキスト配置**: 中央から左側に配置（右側はSNSでトリミングされる可能性あり）
3. **コントラスト**: 背景と文字のコントラスト比4.5:1以上

## 各ページ用バリエーション

必要に応じて、以下のページ用にOGP画像を作成：

| ページ | ファイル名 | 追加テキスト |
|--------|-----------|-------------|
| トップページ | ogp.png | （基本版） |
| サービス | ogp-service.png | 「サービス一覧」 |
| 出生証明書 | ogp-birth.png | 「出生証明書取得」 |
| 婚姻証明書 | ogp-marriage.png | 「婚姻証明書取得」 |
| CENOMAR | ogp-cenomar.png | 「独身証明書取得」 |
| NBI | ogp-nbi.png | 「無犯罪証明書取得」 |
| 料金 | ogp-price.png | 「料金表」 |

## ツール推奨

- **Canva**: 無料でOGP画像作成可能
- **Figma**: デザインチームがいる場合
- **Adobe Express**: Adobeユーザー向け

```

├── ogp.png              # メインOGP画像
├── ogp-service.png      # サービスページ用
├── ogp-birth.png        # 出生証明書ページ用
├── ...
├── favicon.ico          # ファビコン
├── favicon-32x32.png    # 32x32ファビコン
├── favicon-16x16.png    # 16x16ファビコン

```
