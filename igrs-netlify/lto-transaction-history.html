LTO運転経歴証明書（Transaction History）取得代行

LTO運転経歴証明書取得代行とは、フィリピン陸運局（LTO）発行の「運転免許の履歴証明書」を代理取得し、認証・発送するサービスです。

日本の運転免許への書き換え（外免切替）で必須となるLTO証明書を、DFAアポスティーユ付きで最短3週間程度で手配します。

外免切替（フィリピン免許から日本免許への切り替え）で最も重要な書類です。IGRSなら複雑な手続きを完全代行します。

なぜIGRSが選ばれるのか

1. LTO本局へ直接申請

セブ島LTO本局へスタッフが直接出向いて申請します。オンラインシステムに反映されていない古い記録も調査可能です。

2. 外免切替のノウハウ

日本の運転免許試験場で求められる記載事項（免許取得日、更新履歴など）を熟知。不備による再取得を防ぎます。

3. 日本法人の安心感

運営は日本法人です。個人ブローカーに依頼するリスク（連絡不通、金銭トラブル）を回避できます。

4. 進捗の可視化

「LTO取得完了」「DFA認証中」などのステータスを週次でご報告。進捗が見えるから安心です。

LTO Transaction Historyとは何か

LTO Transaction History（運転経歴証明書）は、フィリピン陸運局（LTO: Land Transportation Office）が発行する、運転免許の取得日・更新履歴・有効期間などを証明する書類です。

主な用途

  • 外免切替 - フィリピン運転免許を日本の運転免許に切り替える手続き
  • 運転経歴の証明 - 海外での運転経験を証明する場合

外免切替とは

外国で取得した運転免許を日本の運転免許に切り替える手続きです。各都道府県の運転免許試験場で申請します。フィリピンで免許を取得・保有していた期間が3ヶ月以上あることの証明としてLTO Transaction Historyが求められます。

納期（LTO → DFA → 発送）

納期は「取得のみ」か「Apostille（アポスティーユ）付き」か、さらに国際発送の方法で変動します。

運転経歴証明書の申請・発行

DFA（フィリピン外務省）Apostille処理

外免切替では通常必要

国際発送

DHLで日本へ発送

最短は手配開始から3〜4週間

ただし、LTOの窓口状況、DFAの混雑、通関状況により前後します。

よくあるご質問（FAQ）

Q1. 標準の納期はどのくらいですか？

A: LTO取得からDFA認証、国際発送まで含めて約4週間が目安です。

Q2. 原本は必要ですか？

A: 免許証の原本は不要です。鮮明な写真（表・裏）とパスポートコピーがあれば申請可能です。

Q3. 日本語訳は対応していますか？

A: はい、別途オプションにて日本語翻訳（翻訳証明付き）も承っております。

Q4. 申請に必要な情報は何ですか？

A: 氏名、生年月日、免許証番号、発行日などの情報が必要です。免許証の写真があれば確認可能です。
