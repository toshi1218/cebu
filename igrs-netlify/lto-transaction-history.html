サービス内容

フィリピン陸運局（LTO）発行の運転経歴証明書（Transaction History）の取得を代行。必要に応じてDFAアポスティーユと国際発送まで一括対応します。

標準納期：3〜4週間

LTO Transaction Historyとは何か

LTO Transaction History（運転経歴証明書）は、フィリピン陸運局（LTO: Land Transportation Office）が発行する、運転免許の取得日・更新履歴・有効期間などを証明する書類です。

主な用途

  • 外免切替 - フィリピン運転免許を日本の運転免許に切り替える手続き
  • 運転経歴の証明 - 海外での運転経験を証明する場合

外免切替とは

外国で取得した運転免許を日本の運転免許に切り替える手続きです。各都道府県の運転免許試験場で申請します。フィリピンで免許を取得・保有していた期間が3ヶ月以上あることの証明としてLTO Transaction Historyが求められます。

納期（LTO → DFA → 発送）

納期は「取得のみ」か「Apostille（アポスティーユ）付き」か、さらに国際発送の方法で変動します。

LTO（フィリピン陸運局）での取得

運転経歴証明書の申請・発行

DFA（フィリピン外務省）Apostille処理

外免切替では通常必要

国際発送

EMS/DHLで日本へ発送

最短は手配開始から3〜4週間

ただし、LTOの窓口状況、DFAの混雑、通関状況により前後します。

よくあるご質問（FAQ）

Q1. LTO Transaction Historyとは何ですか？

フィリピン陸運局（LTO）が発行する運転免許の取得・更新履歴を証明する書類です。日本での外免切替に使用されます。

Q2. 標準の納期はどのくらいですか？

LTOでの取得からDFAアポスティーユ、国際発送まで含めて3〜4週間が目安です。

Q3. 外免切替とは何ですか？

海外で取得した運転免許を日本の運転免許に切り替える手続きです。運転免許試験場で行います。

Q4. DFAアポスティーユは必要ですか？

外免切替では通常、DFAアポスティーユ付きが求められます。セットでのご依頼をおすすめします。

Q5. フィリピンの運転免許証原本は必要ですか？

原本は不要です。免許証の写し（表・裏）があれば手続きを進められます。

Q6. 日本語訳は必要ですか？

外免切替では日本語訳が必要です。翻訳サービスも対応可能ですのでご相談ください。

Q7. 進捗報告はもらえますか？

はい、基本的に週1回程度、または重要な進展があった際にメールでご報告します。

Q8. 急ぎの対応は可能ですか？

外免切替の予定日など期限がある場合、可能な限り優先対応いたします。お問い合わせ時にお知らせください。
